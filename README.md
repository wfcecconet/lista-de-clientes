# Lista de Clientes
O projeto consiste na implementação de uma lista para tratamentos de dados em linguagem 'C'. Programa desenvolvido para a disciplina de **Estruturas de Dados 1**.

## Funcionalidades:

- Adicionar cliente
- Remover cliente
- Listar clientes
- Buscar RG
- Ordenação de dados (Selection, Insertion, Bubble, Shell, Quick, Merge)

## Tecnologias:

- Linguagem 'C'
- Estrutura de Dados do tipo Lista Sequencial (Array dinâmico)
- Ferramentas: Compilador GCC/Terminal, Codeblocks
